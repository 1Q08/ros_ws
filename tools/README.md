# README

## 必备工具

> **ROS1/ROS2：**
>
> - [PlotJuggler](https://github.com/PlotJuggler/PlotJuggler)：可视化时间序列的工具。
> - RQt：基于 Qt 图形框架为 ROS 开发的一套图形用户界面工具集和插件开发框架。
> - RViz：RViz（ROS Visualization）可用于实时显示机器人传感器数据、状态信息、环境地图和规划路径。

---

## 自己写的脚本

> - sysinfo.sh：显示必要的系统信息，并提供常用软件安装选项以及系统更新与清理功能。
> - set_fan_profile.sh：目前仅支持切换风扇散热模式。
